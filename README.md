# Week 3 Day 1 - Machine Learning Fundamentals

## Objective

The objective of this assignment is to understand the complete Machine Learning workflow and build a predictive model using Linear Regression. The model predicts student marks based on study hours, attendance, and assignment completion.

## Dataset

The Student Performance Dataset contains the following features:

* Study_Hours
* Attendance
* Assignments

Target Variable:

* Marks

## Tasks Completed

### Part 1: Dataset Exploration

* Loaded the dataset using Pandas
* Displayed first and last 5 rows
* Checked dataset shape
* Displayed column names
* Checked data types
* Generated summary statistics

### Part 2: Features and Label Identification

* Identified features (Study_Hours, Attendance, Assignments)
* Identified label (Marks)

### Part 3: Machine Learning Workflow Documentation

* Data Collection
* Data Cleaning
* Feature Selection
* Train-Test Split
* Model Training
* Testing
* Evaluation
* Prediction

### Part 4: Train-Test Split

* Split dataset into 80% training data and 20% testing data

### Part 5: Linear Regression Model

* Trained Linear Regression model
* Obtained model coefficients and intercept

### Part 6: Predictions

* Predicted marks for new student data

### Part 7: Actual vs Predicted Comparison

* Compared actual marks with predicted marks
* Calculated prediction differences

### Part 8: Model Evaluation

* Calculated Mean Absolute Error (MAE)
* Calculated R² Score

### Part 9: Data Visualization

Created the following Plotly visualizations:

* Study Hours vs Marks
* Attendance vs Marks
* Assignments vs Marks
* Regression Line Visualization
* Predicted Marks Trend

### Part 10: Research Activity

* Machine Learning
* Linear Regression
* Features
* Labels
* Training Data vs Testing Data
* Real-Life Applications of Machine Learning
